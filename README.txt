Boruto Fully Colored Website

Repository name: boruto-fully-colored
Site URL: https://borutoink.github.io/boruto-fully-colored/

How to update chapters:
1. Open your repository on GitHub.
2. Upload your new chapter pages into the corresponding folder under /chapters/.
   Example: upload images for chapter 68 into /chapters/chapter68/.
3. Replace the placeholder cover (comingsoon.png) in /assets/images/ with your chapter's cover.
4. Edit index.html to update chapter names and links if needed.
5. Commit your changes. The site updates automatically.

Enjoy! :)
